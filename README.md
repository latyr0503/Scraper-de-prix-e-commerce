# Scraper de prix e-commerce

Bienvenue dans le projet **Scraper de prix e-commerce** ! Ce projet a pour but de suivre l'évolution des prix de vos produits favoris et de vous alerter en cas de baisse.

## Fonctionnalités actuelles

- **Affichage des produits** : Parcourez le catalogue fictif de produits disponibles.
- **Recherche** : Trouvez un produit spécifique par son nom.
- **Filtrage** : Affichez uniquement les produits appartenant à une catégorie spécifique.
- **Tri** : Triez les produits par prix pour repérer les meilleures offres.
- **Comparaison** : Comparez l'ancien et le nouveau prix d'un produit et recevez une alerte si le prix passe sous un seuil défini.

## Comment utiliser le programme

1. Assurez-vous d'avoir Python installé sur votre machine.
2. Lancez le programme interactif dans votre terminal avec la commande suivante :

```bash
python main.py
```

3. Un menu s'affichera, vous n'avez plus qu'à choisir une option en entrant le numéro correspondant !
